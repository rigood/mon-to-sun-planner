## 1. 프로젝트 소개

월요일(Mon)부터 일요일(Sun)까지 주간 일정을 한 눈에 쉽게 확인하고 관리할 수 있는 플래너입니다.

- 요일별 Task 추가, 수정, 완료 여부 표시, 삭제
- `드래그 앤 드롭`으로 요일 간 Task 이동
- 화살표(<,>) 버튼을 통해 주간 이동
- 달력 아이콘에서 날짜 선택 시 해당 날짜로 이동
